Questão 01

Item A

Pseudocódigo


        mochila(c, p, v, n)
        
        Entrada: uma mochila que suporta C kg, e temos um conjunto com n itens 
        onde cada item i tem peso p[i] e valor v[i].
        
        Saída: subconjunto S de itens de valor máximo e que cabe na mochila.

            bits ← []
            for i de 1 até n faça:
                bits[i] ← 0
            i ← 0
            maior ← 0
            S ← []

            enquanto (i < (2^n - 1)) faça:
                bits ← proxima_sequencia_bits(bits)
                soma_de_pesos ← 0
                soma_de_valores ← 0
                aux ← []

                for k de 1 até n faça:
                    se (bits[k] == 1) então:
                        soma_de_pesos ← soma_de_pesos + 1
                        soma_de_valores ← soma_de_valores + 1
                        aux.append(k)
                se (c >= soma_de_pesos) então:
                    se maior < soma_de_valores então:
                        maior ← soma_de_valores
                        S ←  aux.copy()
            retorne S

Item B

Complexidade do algoritmo

Temos que: \(O(n) + O(2^n) + O(n)\) = \(O(n) + 2*O(n*2^n) = O(n*2^n)\).

Logo a Complexidade é: \(O(n * 2^n)\)
