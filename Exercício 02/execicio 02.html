• Questão 01:

A) \(f(n) = n^2 + 3n^2* \log n\)

Resposta:

1º termo = \(n^2 => c = 1, b^a = 1, d = 2, e = 0 \)

2º termo = \(3n^2* \log n => c=3, b^a = 1, d=2, e=1\)

Logo, \(f(n) = 3n^2* \log n => n^2* \log n \) por ter o maior valor de \(e\).

\(f(n) ∈ Θ(n^2* \log n) => \) que é do tipo polinomial. \( (n^{Θ(1)}) \).


B) \(f(n) = 5n^3 - 17n + 4\)

Resposta:

1º termo = \( 5n^3 => c = 5, b^a = 1, d = 3, e = 0\)

2º termo = \( 17n => c = 17, b^a = 1, d = 1, e = 0 \)

3º termo = \( 4 \)

Logo, \(f(n) = 5n^3 => n^3 \) por ter o maior valor de \(d\).

\(f(n) ∈ Θ(n^3) => \) que é do tipo polinomial. \( (n^{Θ(1)}) \).


C) \(f(n) = 7^{3 * \log_{2} n}\)

Resposta:

1º termo = \( 7^{3 * \log_{2} n} => c=7, b^a =3 * \log_{2} n, d = 1, e = 8\)

Logo, \(f(n) = 7^{3 * \log_{2} n}\)

\( f(n) ∈ Θ(7^{3 * \log_{2} n}) => Θ(2^{Θn}) =>\) que é do tipo exponencial. \( (2^{Θ(n)}) \).


D) \(f(n) = \) { \( 1 \) se \(n\) é impar, \(2\) se \(n\) é par}

Resposta:

1º termo = \(1\) ou \(2\)

Logo, \( f(n)∈ Θ(1) => \) como \(f(n)\) só assume os valores 1 e 2, que é do tipo constante. \((Θ(1))\).


E) \(f(n) = 2* 2^n * n^2 * \log_{2} n - 7n^8 + 7* 3^n / n^2\)

Resposta:

1º termo = \( 2* 2^n * n^2 * \log_{2} n => c = 2, b^a = 2, d = 2, e = 1 \)

2º termo = \( 7n^8 => c = 7, b^a = 1, d = 8, e = 0\)

3º termo = \( 7* 3^n / n^2 => c = 7, b^a = 3, d = -2, e=0\)

Logo, \( f(n) = 7* 3^n / n^2 \) por ter o maior valor de \(b^a\)

\( f(n) ∈ Θ( 3^n / n^2) => \) que é do tipo exponencial decrescente.

Logo, \( f(n) = 3^{4n} \), por ter o maior valor de \(b^a\).

\(f(n) ∈ Θ(3^{4n}) => \) que é do tipo exponencial. \( (2^{Θ(n)}) \).
