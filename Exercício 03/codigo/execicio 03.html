B)- RESPOSTA

Bubble_sort:

Assumimos que o tempo de execução do algoritimo é \(T(n)= c*n^2\), para encontrar a costante c, utilizamos o ultimo ponto do gráfico em relação com a entrada, ou seja \(n\) vale 51200 e o tempo vale 236,31.

\(T(n) = c*n^2 = 236,31 = c * 51200^2 => c = 9,14*10^{-8} \)

\(T(n) = 9,14*10^{-8} * n^2\)

Testando para o caso da entrada de tamanho 51200, se obteve o resultado: \(T(51200) = 236,31\) que é próximo ao tempo gasto pelo computador durante a execução, que foi de \(236.31884241104126\).

Testando para a entrada de tamaho 80000, se obeteve o seguinte resultado: \(T(80000) = 584.96\) que é próximo ao tempo gasto pelo computador durante a execução, que foi de \(580.7523713111877\).

\(T(n) = c*n \)

Subisituindo \(T(n)\) por o maior tempo de execução e \(n\) por a maior entrada, para encontrar o valor da constante c, temos que: \(94,58 = c * 2^{1*30} => c= 8,80*10^{-8}\), Logo a expressão para previsão do tempo de execução em função do valor da variável \(n\) de entrada é \(T(n) = 8,80*10^{-8} * 2^n \).
