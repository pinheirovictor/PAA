  • Questão 04:
  • Prova por contradição

Definição: este metodo de prova é feito assumindo-se como verdade o contrário do que se quer provar, realizar uma prova direta e então chegar a uma contradição.

Demostração:

"Se \(3n + 2\) é ímpar, então \(n\) é ímpar"

p ← "Se \(3n + 2\) é ímpar"

q ← "então \(n\) é ímpar"

Para provarmos por contradição, assumimos que p e ~q são verdadeiras, logo \(3n + 2\) é ímpar e \(n\) não é impar.

Como \(n\) não é ímpar, logo \(n\) é par.

Se \(n\) é par então \(3n + 2\) também é par. como \(n\) é par, pode ser escrito como \(n = 2k\) onde \(k\) pertence aos inteiros.

isso implica que \(3n + 2 = 3(2k) + 2 = 6k + 2 = 2(3k+1).\) Como \(3n + 2 = 2t,\) em que \( t = 3k + 1, 3n + 2\) é par.

Como a setença \(3n + 2\) é par é o mesmo que ~p, pois um inteiro só é par sse não for ímpar. Como ambas p e ~q são verdadeiras, temos uma contradição, logo \(3n + 2\) é ímpar, então \(n\) é ímpar.

Também se pode demonstrar por contradição que p-> q é verdadeira, assumindo que p e~q são verdadeiras e mostrando que q deve ser também verdadeira. Isso implica que ~q e q são ambas verdadeiras, logo uma contradição.

  • Prova por indução

Definição: A indução matemática pode ser usada para se demonstrar proposições que afirmam que \(P(n)\) é verdade para todos os numeros inteiros positivos, onde \(P(n)\) é uma função proposicional.

indução é divida em duas partes, uma parte chama passo base, onde se motrsa que \(P(1)\) é verdade e o passo indutivo, que se mostra para todos interios, onde se \(P(k)\) é verdade então \(P(k+1)\) também é verdade

Demonstração:

\(n < 2^n\) para todos os inteiros positivos \(n\).

Passo Base: \(P(1) = 1< 2^1 = 2\), logo \(P(1)\) é verdade!

Passo indutivo: assumimos que \(P(k)\) é verdade para um inteiro k positivo, ou seja a hipótse de indução é \(k < 2^k\).

Se \(P(k)\) é verdade, logo assumimos que \(P(k + 1)\) é verdade.

Logo \(k + 1 < 2^{k+1}\) é verdade. Precisamos mostrar que se \(k < 2^k\) então \(k + 1 < 2^{k+1}\)

Se somarmos 1 aos dois lados de \(k < 2^k\), temos que \(1 <= 2^k\), isso nos diz que:

\(k + 1 < 2^k + 1 <= 2^k + 2^k = 2 * 2^k = 2^{k+1}\)

Isso nos diz que \(P(k + 1)\) é verdade, ou seja \(k + 1 < 2^{k+1}\), com base na H.I. de que \(P(k)\) seja verdade.

Logo concluí-se que \(n < 2^n\) é verdade para todos os inteiros positivos \(n\).
